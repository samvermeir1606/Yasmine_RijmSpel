Minigame 3 gebruikt nu een eigen map met dezelfde paar-naamstructuur als minigame 2.

Structuur:
- pairs/index.js
- pairs/schaap - aap.png
- pairs/schaap - aap.mp3
- pairs/aap - schaap.png
- pairs/aap - schaap.mp3

Naamregels:
- Elk kaartje krijgt de naam: WOORD - PARTNER
- Voor elk rijmpaar maak je dus 2 kaartjes:
- `schaap - aap`
- `aap - schaap`

Belangrijk:
- Deze map is volledig los van minigame 2
- Je kan hier later andere woorden zetten zonder minigame 2 te veranderen
- Voeg nieuwe kaartjes ook toe aan `pairs/index.js`
- Afbeeldingen mogen `.png`, `.jpg`, `.jpeg`, `.webp`, `.gif` of `.svg` zijn
- Audio gebruikt momenteel `.mp3`
- Het memory-spel gebruikt uit deze map 3 paren per ronde
